Peticiones desde el shell

Utilice las siguientes herramientas para descargar un fichero y navegar desde el shell:
  • curl: es un proyecto de software consistente en una biblioteca (libcurl) y un intérprete de comandos (curl) orientado a la transferencia de archivos.
  • wget: es una herramienta libre que permite la descarga de contenidos desde servidores web de una forma simple. Su nombre deriva de World Wide Web
  • lynx: es un navegador web y cliente de gopher en modo texto.
  • links: es un navegador web de código abierto en modo texto —y gráfico a partir de su versión 2 en modo terminal
